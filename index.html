Fuel Efficiency

The mtcars-predictor application at shinyapps models fuel efficiency in miles per gallon using an automobile's weight and number of engine cylinders as predictors.

The interface is a simple one. A user inputs the weight in thousands of pounds and the number of cylinders (4 ,6, or 8) and the application immediately computes the estimated fuel efficiency and shows details about the regression analysis.

Regression Model

Before modeling the relationship, a transformation is applied to factorize the cyl feature (number of cylinders):

data(mtcars)

mtcars$cyl = factor(mtcars$cyl, 
  levels=c(4,6,8))

model = lm( mpg ~ wt + cyl, data=mtcars )

The model appears to be quite effective at accounting for the variation in the outcome with an adjusted R squared of 0.82.

Exploratory Analysis

The model was derived from an exploratory analysis of the data that suggested weight and number of engine cylinders might effectively explain differences in the fuel efficiency.

It was discovered that the mean weight of the most efficient vehicles (over 25 MPG) was only ~1800lbs while that of all others was ~3500lbs. Similarly, all of the most efficient vehicles had only 4 engine cylinders.
